# Contraption Zac (2D Platformer Rebuild: First 10 Levels) 

## Overview
As part of a software engineering project stduents recreated the first 10 levels of the online game Contraption Za.c The goal of this project was to recreate the mechanics and feel of a 3D platformer experience in a 2D environment while working collaboratively as a team of three students.

This project emphasized teamwork, software design principles, and building a structured game architecture from scratch.

---

## Team Project
This game was developed in a team of 3 students as part of a software engineering course. Responsibilities were divided across gameplay programming, level design, and system architecture.

---

## Gameplay
- Control the main character through platform-based levels
- Move across platforms and avoid obstacles
- Navigate through progressively challenging stages
- Reach the end of each level to progress

---

## Features
- 2D platforming mechanics (movement and collision detection)
- Level-based progression system
- Obstacle interactions
- Structured game loop and modular code design

---

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Game loop architecture
- Basic physics simulation
- Team-based version control (Git)
